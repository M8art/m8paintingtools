Privacy Policy

M8 Painting Tools is a set of painting study tools designed for artists. This page explains what information is processed when you use the website, installed web app, or Google Play version.

App and developer

App name: M8 Painting Tools.

Developer: M8art / Jan Matěják.

Privacy contact: matejakjan@gmail.com.

Uploaded images for tool use

For normal use of the painting tools, uploaded images are processed locally in the user's browser or app webview. These images are used to calculate value, composition, color, and related visual readings on the user's device.

Normal tool uploads are not uploaded to our server and are not stored by us as part of the standard analysis workflow.

Quick Analysis AI image processing

When a user chooses Quick Analysis in Google Play mode, the uploaded image may be sent securely to a backend AI service so the requested painting critique can be generated.

These images are used only to generate the requested analysis. They are not sold and are not stored permanently by M8 Painting Tools.

User accounts

The app does not require user accounts. Users can use the normal tools without creating a login.

Contact and support forms

Contact forms may collect name, email, and message if the user voluntarily submits them. Report forms may collect the report reason, optional email, and message text that the user chooses to provide.

This voluntarily submitted data is used only to respond to the user, handle the request, review reported analysis output, or provide the requested support.

Analytics

The website version may use Google Analytics to understand basic usage and improve the tools. Analytics, when active, is used for aggregate usage measurement and does not require a user account.

If analytics providers or analytics behavior change later, this policy must be updated before those changes are used.

Payments and Google Play

The web version may use external payment providers for optional unlocks or services where applicable. Payment details are handled by the payment provider and are not stored by M8 Painting Tools.

The Google Play version does not currently process in-app payments.

Sharing and selling data

Data is not sold.

Data is not shared except with service providers needed to run the website or app, including hosting, email or contact handling, form handling, analytics where applicable, and payment processing where applicable.

Retention and deletion

User-submitted contact or feedback data is kept only as long as needed to respond to the user, provide the requested service, maintain records for that request, or meet legal or security requirements.

Users can request deletion of submitted contact or feedback data by emailing matejakjan@gmail.com.

Public access

This Privacy Policy is publicly accessible and is not editable by users.
