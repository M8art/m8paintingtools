Phone Setup

How to install M8 Painting Tools on your phone

Open the website once, add it to your home screen, and the M8 Tools icon will sit on your phone like a normal app. It is still a web app, so it stays easy to update while I keep improving the mobile version.

M8 Tools Add to Home Screen

Android

Install from Chrome

  1. Open m8paintingtools.com in Chrome.
  2. Tap the three dots in the top-right corner.
  3. Choose Add to Home screen or Install app.
  4. Confirm the install.
  5. The M8 Tools icon will appear on your phone like a normal app.

iPhone / iPad

Add from Safari

  1. Open m8paintingtools.com in Safari.
  2. Tap the Share button at the bottom of the screen.
  3. Choose Add to Home Screen.
  4. Tap Add.
  5. The M8 Tools icon will appear on your iPhone or iPad like an app.
